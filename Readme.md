# go chain


## commond

## run go chain 

`cd goblockchain`

`go build`

`./goblockchain`

## Usage

### create a blockchain

`goblockchain.exe createblockchain -address XXX`

### get balance in a blockchain

`goblockchain.exe getbalance -address XXX`

### sent a coin to address

`goblockchain.exe send -from XXX -to Gabriel -amount 6`
